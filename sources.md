# Description
## Presentation
- **NetPractice** is a practical introduction to the fundamental concepts of computer networking.
- Through a series of hands-on exercises, the project focuses on understanding and applying the basics of
  **TCP/IP** configuration in small-scale networks.

## Goal
- The main goal of this project is to learn how devices communicate within a network by correctly configuring:
  - **IP addresses**.
  - **Subnet masks and logic**.
  - **Default gateways**.
- In order to achieve this goal, we learned:
    - How hosts are connected through a **router**.
    - How **subnets** are structured.
    - How proper **addressing** ensures successful communication between different network segments.

## Overview
- The project is divided into 10 progressive levels, each representing a specific networking scenario to
  analyze and configure.
- All exercises are completed within a **simulated networking environment** accessible through a web
  browser.

# Instructions
## Installation details
To be able to run the simulated networking environment:
1. Download the file attached to the project's page.
2. Extract the files into any folder of your choice.
3. In this folder, run the `run.sh` file. It will launch a web server and open your preferred web browser to the dedicated page.
4. Once the interface is open, you can select one of these two modes:
  - Training mode:    Practice by entering your login in order to use your personal configuration.
  - Evaluation mode:  Practice on a random configuration (suitable for evaluations).
## Execution details
For each level, you will have to complete an exercise, and new buttons will appear:
- Check again:      To re-evaluate your solution.
- Get my config:    To export your configuration (DO NOT FORGET THIS PART!!).
- Next level:       To proceed to the next level (next exercise).
## Submission details
For this project, the submission requirements are:
- This README file.
- For each level successfully completed, the exported configuration file **must be placed at the project repository root** _(this file is generated by clicking on the **Get my config** button)_
